------------------------Detección de errores-------------------

1. Error de compilación: En la condición while (salir = false) se usa asignación (=) en lugar de comparación.
   Esto provoca que la condición siempre sea falsa y genera comportamiento incorrecto.

2. Error lógico: La condición del while produce un ciclo infinito.

3. Error lógico: En el for (int i = 0; i <= n; i++) se ejecuta una iteración adicional.
   Debería ser i < n.

4. Error de compilación: El if (nota >= 0 && nota <= 10); contiene un punto y coma al final.
   Esto anula la condición

5. Error de compilación: El else queda sin relación directa con el if debido al punto y coma.

6. Error lógico: cantidad++ se ejecuta aunque la nota sea inválida

7. Mala práctica: No se usan llaves {} en estructuras if-else

8. Error lógico: Falta break al final del case 1, provocando caída al case 2

9. Error lógico: double promedio = total / cantidad realiza división entera

10. Error lógico: No se valida si cantidad es 0 antes de dividi

11. Error de compilación: else if (promedio >= 7); tiene punto y coma

12. Error lógico: "Regular" se imprime siempre por el punto y coma

13. Error de compilación: while (i < cantidad); contiene punto y coma que rompe el ciclo.

14. Error lógico: En estadísticas se usa total / cantidad en lugar de evaluar cada nota.

15. Error lógico: No se almacenan las notas en una estructura para análisis individual.

16. Error lógico: Falta break en case 4.

17. Error lógico: Falta break en case 5.

18. Error lógico: El default puede ejecutarse mal por falta de breaks.

19. Mala práctica: No se valida rango de opción ingresada.

20. Mala práctica: El reinicio no contempla limpiar estructura de almacenamiento.
